# ReactExpressTemplate

ReactExpressTemplate is a small 2018 React plus Express boilerplate with an Express `server.js` and a React client folder.

## What It Is

This repo provides a simple full-stack JavaScript template. The root package runs an Express server, can run the client through `npm start --prefix client`, and has a `dev` script that starts server and client together through `concurrently`.

## Current Status

The root `package.json` lists Express 4.16.4, Nodemon, and Concurrently. The repo contains a `client/` app and root `server.js`.

No local build or runtime test was run for this README.

## Tech Stack

- Node.js
- Express
- React client folder
- npm scripts

## Limitations
